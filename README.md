# 📚 Book Store Management System

A simple command-line based book store management system built using Python.

## Features

* Add new books to inventory
* Delete books using ISBN
* View available inventory
* Sell books and update stock
* Track total sales revenue

## Concepts Used

* Basic data structures (lists, dictionaries)

## How to Run

1. Make sure Python is installed
2. Download the project files
3. Run the program:

   ```bash
   python main.py
   ```

## 📌 Notes

* Inventory is stored in memory (not saved permanently) KEEP THIS IN MIND PLEASE.
* When stock reaches 0, the book is removed automatically
